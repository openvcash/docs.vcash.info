### getincentiveinfo ###

Returns information about the current incentive state.

### Example ###

```
getincentiveinfo

{
    "walletaddress": "VwhF7PwsYWNkuFpk7KQm3xSTjv79KJWfYX",
    "collateralrequired": 9999,
    "collateralbalance": 10319,
    "networkstatus": "open",
    "votecandidate": true,
    "votescore": 1186
}
```
